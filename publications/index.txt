In this study, data representation techniques are applied to the task of real-time boiler fault detection. A CNN-GRU-Attention (GCA) model is designed, where raw time-series data are first transformed using Gramian Angular Field (GAF) to enable effective data representation learning. Subsequently, 1D temporal features and 2D spatial features are extracted through Gated Recurrent Unit (GRU) and Convolutional Neural Network (CNN) branches, respectively.
